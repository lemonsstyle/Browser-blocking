# Result Keyword Filter

[English](README.md) | 简体中文

一个最小可用的 Chrome / Edge 扩展，用来隐藏搜索结果页中标题包含指定关键词的条目。

## 默认屏蔽词

默认关键词列表为空。请在扩展弹窗里添加你自己的关键词。

## 默认编号样式规则

- 启用过滤
- 字母最少 `3`
- 字母最多 `5`
- 分隔符 `可无、横杠或空格`
- 数字最少 `3`
- 数字最多 `4`

## 安装方式

1. 打开 Chrome 或 Edge 的扩展管理页。
2. 开启“开发者模式”。
3. 选择“加载已解压的扩展程序”。
4. 选择当前项目目录。

## 使用方式

1. 打开目标搜索结果页。
2. 点击浏览器工具栏里的扩展图标。
3. 在弹窗顶部选择模板，或点击“新建模板”创建一套新的屏蔽配置。
4. 在“适用网站 / URL 前缀”中填写这个模板要过滤的网站；可点击“填入当前网站”自动带入当前域名，再按需修改。
5. 填写 `weibo.com` 会匹配该域名及其所有子域名；填写完整 URL（如 `https://weibo.com/u/`）则只匹配此前缀。
6. 在弹窗里编辑屏蔽词，支持用逗号、空格、分号或换行分隔。
7. 在“编号样式过滤”里调整字母位数、分隔符规则、数字位数。
8. 点击“保存当前模板”后，配置会写入浏览器存储。
9. 页面中命中关键词或编号样式规则的结果会被自动隐藏，动态加载的新结果也会继续过滤。

## 模板规则

- 扩展会根据当前页面网址自动选择模板；没有填写适用网站或 URL 前缀的模板不会过滤任何页面。
- 若有多个模板同时匹配，优先使用范围更具体的 URL 前缀或域名；完全相同时使用最近保存的模板。
- 新建模板会复制当前表单里的关键词和编号样式配置。
- 切换模板只会切换弹窗里正在编辑的模板，实际过滤由网址匹配决定。
- 旧版保存过的全局关键词会自动作为“默认模板”读取，但需要先填写适用网站才会继续过滤。

## 实现思路

- `content.js` 会先按当前页面的域名或 URL 前缀选择模板，再扫描列表项、表格行、卡片、标题链接等常见结果容器。
- 如果容器文本命中屏蔽词，就直接 `display: none`。
- 用 `MutationObserver` 监听异步加载内容，保证翻页或懒加载后仍然生效。
